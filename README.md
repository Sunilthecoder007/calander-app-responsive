# Calendar App Responsive version

A modern calendar app built with React, TypeScript, and Vite.

## Technologies
- React 18
- TypeScript
- Vite
- TailwindCSS
- Lucide React (icons)

## Prerequisites
- Node.js (LTS)
- npm/yarn
